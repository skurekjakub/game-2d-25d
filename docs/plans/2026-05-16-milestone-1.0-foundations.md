# Milestone 1.0 — Foundations Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Walkable empty bounded arena with all foundational plumbing (git, tests, autoloads, debug, input) in place. After this milestone you can launch the game, move a placeholder circle around an empty room with WASD, and run automated tests.

**Architecture:** Three autoloads (`Game`, `EventBus`, `Debug`) form the global service layer. One `Arena` scene contains a `Player` (CharacterBody2D) and bounded walls. Tests live in `tests/` and run headlessly via the chosen test framework. Follows the file layout in `docs/design/2026-05-16-vampire-survivors-clone-design.md` §5.

**Tech Stack:** Godot 4.6.2, GDScript (statically typed), GdUnit4 for testing.

**Phase 1 milestone roadmap (this plan = 1.0 only):**

| # | Deliverable |
|---|---|
| **1.0** | **Foundations: walkable empty arena, tests run, autoloads exist** (this plan) |
| 1.1 | One enemy walks toward player; one weapon kills it; first damage system |
| 1.2 | SpawnerDirector + SpawnSchedule; survive a 1-min wave |
| 1.3 | XP gems + level-up dialog; multi-weapon selection |
| 1.4 | All 5 weapons + 5 normal enemy types content-complete |

| 1.6 | Main menu, pause, death, victory screens; full game lifecycle |

- [ ] **Step 3: Implement Game**

Create `bootstrap/game.gd`:

```gdscript
extends Node

class RunState:
	var hp: float
	var max_hp: float
	var xp: int
	var level: int
	var time_elapsed: float

var run_state: RunState


func _ready() -> void:
	run_state = RunState.new()


func start_run() -> void:
	run_state = RunState.new()
	run_state.max_hp = 100.0
	run_state.hp = run_state.max_hp
	run_state.xp = 0
	run_state.level = 1
	run_state.time_elapsed = 0.0
	EventBus.run_started.emit()


func xp_needed(for_level: int) -> int:
	return 5 + for_level * 3


func add_xp(amount: int) -> void:
	run_state.xp += amount
	EventBus.xp_gained.emit(amount)
	while run_state.xp >= xp_needed(run_state.level):
		run_state.xp -= xp_needed(run_state.level)
		run_state.level += 1
		EventBus.level_up.emit(run_state.level)
```

Note: tests instantiate Game directly via `.new()`, which doesn't trigger autoload's `_ready`. That's why `start_run()` (re)initializes `run_state` explicitly.

Edit `project.godot`. Append to the `[autoload]` section:

```

Order matters: EventBus must come before Game in the file (Game references EventBus).

class_name Player
extends CharacterBody2D

@export var speed: float = 200.0


static func compute_velocity(input_vector: Vector2, speed: float) -> Vector2:
	if input_vector == Vector2.ZERO:
		return Vector2.ZERO
	return input_vector.normalized() * speed


func _physics_process(_delta: float) -> void:
	var input_vector := Vector2(
		Input.get_axis("move_left", "move_right"),
		Input.get_axis("move_up", "move_down"),
	)
	velocity = compute_velocity(input_vector, speed)
	move_and_slide()
```
