# Milestone 1.3 — Player HP, Contact Damage, Run-end on Death — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:subagent-driven-development` (recommended) or `superpowers:executing-plans` to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking. The bespoke `godot-implementer` subagent (`.claude/agents/godot-implementer.md`) is the right fit for individual tasks here; the adversarial `rubber-duk` subagent reviews each task before commit.

**Goal:** Close the combat loop the other way — enemies tick contact damage to the Player; HP-zero emits `EventBus.run_ended(false)`.

**Architecture:** Reuse `HealthComponent` on the Player (relocate the file out of the enemy-only directory first). Each enemy gets a child `Hurtbox` (`Area2D`) plus a `ContactDamageComponent` that ticks `EnemyData.contact_damage` to overlapping bodies once per second (fire-on-enter, reset-on-exit, per-body timer dict). On Player's `HealthComponent.died`, the Player goes inert and emits `run_ended(false)`. RunState loses `hp`/`max_hp`.

**Tech Stack:** Godot 4.6.2-stable, GDScript (static typing mandatory), GdUnit4 tests, Area2D detection, signal-driven composition.

**Design source:** [`docs/design/2026-05-17-milestone-1.3-player-hp.md`](../design/2026-05-17-milestone-1.3-player-hp.md)

## Collision-layer plan (referenced by Tasks 5 + 6)

Currently both Player and Enemy default to `collision_layer = 1, collision_mask = 1`. The enemy's Hurtbox needs to detect ONLY the Player — without the layer split, the hurtbox would `body_entered`-fire for every overlapping enemy (and `_is_damageable` would tick damage between enemies because they each have a HealthComponent).

Final assignments:

| Node | `collision_layer` | `collision_mask` | Reason |
|---|---|---|---|
| Player `CharacterBody2D` | **2** (bit 1) | 1 (bit 0) | Lives on its own layer so hurtbox can target it. Still collides with walls + enemies (both on layer 1). |
| Enemy `CharacterBody2D` | 1 (bit 0) — unchanged | **3** (bits 0+1) | Adds layer 2 to mask so it collides with Player physically. Keeps layer 1 so it still collides with walls + other enemies. |
| Enemy `Hurtbox` (Area2D) | 0 — nobody scans it | **2** (bit 1) | Detects ONLY Player; no false positives on other enemies' bodies. |
| XpGem `Area2D` (existing) | unchanged | unchanged | Verify it still detects Player after Player moves to layer 2 — see Task 8 smoke check. (XpGem.mask must include layer 2; if it was 1, gem pickup will break.) |

The XpGem mask is the one cross-cutting risk. **Task 6 must check and update it if needed.**

**Why now:** TDD green phase. Minimum code to turn Task 3's tests green.

**Files:**

- [ ] **Step 1: Write the implementation**

Create `combat/components/contact_damage.gd`:

```gdscript
class_name ContactDamageComponent
extends Node

## Ticks `damage` to overlapping bodies once per `tick_interval` seconds.
## Bind the hurtbox via `hurtbox_path` (NodePath, resolved in _ready).
## Per-body timers stored in a dict by instance_id — entering an already-tracked
## body resets the timer and fires an immediate tick; exiting clears it.

@export var damage: float = 5.0
@export var tick_interval: float = 1.0
@export var hurtbox_path: NodePath

var _hurtbox: Area2D
var _timers: Dictionary = {}  # { instance_id (int): seconds_until_next_tick (float) }


func _ready() -> void:
	_hurtbox = get_node_or_null(hurtbox_path) as Area2D
	assert(_hurtbox != null, "ContactDamageComponent: hurtbox_path must resolve to an Area2D")
	_hurtbox.body_entered.connect(_on_body_entered)
	_hurtbox.body_exited.connect(_on_body_exited)


func _process(delta: float) -> void:
	if _timers.is_empty():
		return
	# Pass 1: decrement timers.
	for body_id in _timers.keys():
		_timers[body_id] -= delta
	# Pass 2: tick + cleanup (don't mutate while iterating in pass 1).
	for body_id in _timers.keys():
		if _timers[body_id] > 0.0:
			continue
		var body: Node = instance_from_id(body_id) as Node
		if body == null or not is_instance_valid(body):
			_timers.erase(body_id)
			continue
		_apply_tick(body)
		_timers[body_id] = tick_interval


func _on_body_entered(body: Node) -> void:
	if not _is_damageable(body):
		return
	_apply_tick(body)
	_timers[body.get_instance_id()] = tick_interval


func _on_body_exited(body: Node) -> void:
	_timers.erase(body.get_instance_id())


func _is_damageable(body: Node) -> bool:
	return _find_health(body) != null


func _apply_tick(body: Node) -> void:
	var health: HealthComponent = _find_health(body)
	if health == null:
		return
	health.take_damage(damage, get_parent())


static func _find_health(body: Node) -> HealthComponent:
	for child in body.get_children():
		if child is HealthComponent:
			return child
	return null
```

@onready var visual: Polygon2D = $Visual

	add_to_group("player")
	_base_color = _visual.color
	_health.damaged.connect(_on_damaged)
	_health.died.connect(_on_died)


static func compute_velocity(input_vector: Vector2, p_speed: float) -> Vector2:
	if input_vector == Vector2.ZERO:
		return Vector2.ZERO
	return input_vector.normalized() * p_speed


func _physics_process(_delta: float) -> void:
	var input_vector := Vector2(

	EventBus.damage_dealt.emit(null, self, amount)

	tween.tween_property(_visual, "color", _base_color, FLASH_DURATION_SEC)
